How is the Data Stored and accessed.
Data is stored by document in a nosql (hadoop) database. Generally each court decision is a document. The statutory law of each jurisdiction is divided by title which results in about 100-300 documents each organizing several dozen related laws. An API and data definition is supplied so people can leverage the data for machine learning research. A web enabled demo app will also be provided to demonstrate how to reference the data.
